# Allegati da analizzare

Il connettore Gmail non permette di scaricare il contenuto reale degli allegati (foto, PDF, documenti): fornisce solo nome file e tipo.

Quando un'analisi e-mail richiede di vedere davvero un allegato, salva qui il file scaricato da Gmail. Da questa cartella può essere letto direttamente, immagini comprese.

Non è una cartella di archiviazione permanente: i file possono essere rimossi una volta conclusa l'analisi.
