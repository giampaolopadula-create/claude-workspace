# Start

Inizializza la sessione caricando tutto il contesto del workspace.

---

## Istruzioni

### Leggi il contesto

Leggi tutti i file nella cartella `context/` per acquisire il contesto completo.

### Leggi le skill disponibili

Leggi le skill disponibili in `.claude/skills/` per sapere cosa puoi fare.

### Riepilogo

Dopo aver letto tutto, scrivi solo una breve conferma che sei pronto ad aiutarmi. NON mostrare a schermo il contenuto dei file di contesto, NON elencare le skill disponibili, NON fare tabelle o riepiloghi. Solo una frase di conferma.
