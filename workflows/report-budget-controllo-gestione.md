# Workflow — Report sintetico budget/controllo di gestione

Rigore analitico e verifica dell'output: vedi [../context/metodo-di-lavoro.md](../context/metodo-di-lavoro.md).

Quando ricevo dati grezzi (export, Excel, PDF) su budget, occupazione, costi:
1. Salva il file grezzo ricevuto in `Lavoro/Accounting/Controllo di gestione & budget/Dati grezzi/`.
2. Produci un report sintetico leggibile in pochi minuti: numeri principali, confronto budget vs actual (o periodo precedente), anomalie, raccomandazioni.
3. Salva il report in `Lavoro/Accounting/Controllo di gestione & budget/Report/YYYY-MM-tema.md` (o .docx).
