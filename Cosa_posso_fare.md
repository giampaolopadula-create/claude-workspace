COSA POSSO FARE PER TE (Claude Code)
=====================================

Non serve saper programmare per usarmi. Puoi semplicemente parlarmi in
italiano e dirmi cosa ti serve. Ecco un elenco delle cose che posso fare.


1. SCRIVERE E MODIFICARE DOCUMENTI
-----------------------------------
- Creare, leggere e modificare file di testo, Word-like (.txt, .md), CSV, ecc.
- Riassumere, correggere, tradurre o riscrivere testi.
- Organizzare appunti, liste, tabelle, checklist.


2. GESTIRE FILE E CARTELLE SUL TUO COMPUTER
--------------------------------------------
- Cercare file per nome o contenuto.
- Rinominare, spostare, copiare file e cartelle.
- Rinominare in blocco tanti file insieme (es. tutte le foto di una cartella).
- Analizzare cartelle piene di file e dirti cosa contengono.


3. LAVORARE CON DATI (EXCEL, CSV, NUMERI)
-------------------------------------------
- Leggere fogli Excel/CSV e fare calcoli, somme, medie, confronti.
- Creare grafici e tabelle a partire dai tuoi dati.
- Pulire dati disordinati (es. elenchi con errori o duplicati).


4. NAVIGARE INTERNET E IL BROWSER PER TE
-------------------------------------------
- Cercare informazioni sul web.
- Aprire siti, cliccare bottoni, compilare form al posto tuo (con Chrome).
- Fare screenshot di pagine web o leggerne il contenuto.


5. CREARE PAGINE WEB E "ARTIFACT"
-------------------------------------------
- Creare piccole pagine web (es. una calcolatrice, un piccolo gioco,
  un cruscotto con grafici, un volantino) che puoi vedere subito nel browser.
- Aggiornarle facilmente se vuoi cambiare qualcosa.


6. SCRIVERE PROGRAMMI E AUTOMAZIONI (se in futuro ti serve)
-------------------------------------------------------------
- Creare piccoli programmi che automatizzano compiti ripetitivi
  (es. rinominare 500 file, unire più PDF, mandare email automatiche).
- Non devi capire il codice: mi dici cosa vuoi ottenere e io mi occupo
  della parte tecnica.


7. RICERCA E ANALISI
-------------------------------------------
- Rispondere a domande, spiegare concetti in modo semplice.
- Confrontare opzioni (es. "quale scelta conviene di più") e darti un
  consiglio pratico.
- Analizzare documenti lunghi (PDF, relazioni, contratti) e tirarne fuori
  i punti chiave.


8. PIANIFICARE ATTIVITA' RIPETUTE
-------------------------------------------
- Impostare controlli automatici che si ripetono nel tempo
  (es. "controlla ogni giorno se è successo X").
- Ricordarti cose o tenere traccia di attività in corso.


9. MEMORIA
-------------------------------------------
- Posso ricordare informazioni utili tra una conversazione e l'altra
  (es. tue preferenze, contesto di un progetto) se me lo chiedi.


COME CHIEDERMI LE COSE
-------------------------------------------
Non serve un linguaggio tecnico. Basta spiegarmi in modo naturale cosa
vuoi ottenere, come lo spiegheresti a una persona. Se manca qualche
dettaglio te lo chiedo io prima di procedere, specialmente per le azioni
che non si possono annullare facilmente (es. cancellare file, inviare
messaggi, pubblicare qualcosa online).
